# Round Internet Money

A collection of open-source packages and protocol definitions.

This organization currently includes:

- `ethereal-rest` for the Ethereal REST API
- `ethereal-wss` for the Ethereal websocket API
- `pb-dex` for protobuf wrappers and shared types
- `proto-dex` for the underlying protobuf specifications

Go packages can also installed and imported through:

`https://roundinternet.money/<package-name>`

If you're interested in protocol-level work, see the Buf page:

`https://buf.build/round-internet-money/`
